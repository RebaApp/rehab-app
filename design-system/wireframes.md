## Ключевые принципы wireframes:

1. **Сохранение функционала**: Все существующие элементы интерфейса присутствуют
2. **Иерархия информации**: Четкое разделение контента по важности
3. **Доступность**: Минимальные размеры тап-зон 44px
4. **Адаптивность**: Компоненты масштабируются для разных размеров экранов
5. **Консистентность**: Единообразное использование spacing и typography
6. **Фокус на контенте**: Минималистичный подход без лишних элементов

## Компоненты для реализации:

- **Header**: Заголовок с кнопкой закрытия
- **Search Input**: Поле поиска с иконкой
- **Card**: Универсальная карточка для контента
- **Button**: Кнопки разных типов и размеров
- **Tab Navigation**: Переключение между разделами
- **Modal**: Модальные окна для фильтров и форм
- **Bottom Navigation**: Навигация между экранами
- **Image Gallery**: Галерея изображений
- **Rating**: Компонент рейтинга со звездами
- **Tag**: Теги для категорий и услуг
- **Input**: Поля ввода с валидацией
- **Checkbox**: Чекбоксы для множественного выбора
- **Slider**: Слайдер для диапазонов значений
