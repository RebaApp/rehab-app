### Сравнение деталей дизайна:

| Элемент | Консервативная | Прогрессивная |
|---------|----------------|---------------|
| **Заголовки** | Inter Bold, 36px | Inter Bold, 48px, увеличенный letter-spacing |
| **Основной текст** | Inter Regular, 14px | Inter Regular, 16px, line-height: 1.6 |
| **Карточки** | Shadow: 0 2px 8px | Shadow: 0 8px 32px (floating effect) |
| **Spacing** | Стандартный | Увеличенный между элементами |
| **Поиск** | Обычный input | Glassmorphism с backdrop-filter |
| **Кнопки** | Стандартные | С hover effects и анимациями |
| **Теги** | Простые | С rounded corners и subtle background |
| **Навигация** | Стандартная | С underline для активного таба |

### Ключевые отличия прогрессивной концепции:

1. **Больше белого пространства** для дыхания
2. **Увеличенные размеры шрифтов** для лучшей читаемости
3. **Floating эффекты** для карточек и элементов
4. **Glassmorphism** для поиска и модальных окон
5. **Улучшенная типографическая иерархия**
6. **Subtle анимации** и micro-interactions
7. **Современные паттерны** дизайна
